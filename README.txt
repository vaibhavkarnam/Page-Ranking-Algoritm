INTRODUCTION:
------------
Link Analysis and PageRank Implementation

PRE-REQUISITS:
-------------
JRE 1.8.101 or higher

EXTERNAL LIBRARIES USED:
-----------------------
jsoup-1.10.2

ITEMS INCLUDED IN THE ZIP FOLDER
--------------------------------
- jsoup-1.10.2 external JAR which is used in our program.
- Java source files for the program in the folder src
- Executable JAR files Task1 and Task2 in folder jars
- G1_perplexity is the text file containing the perplexity values for Task1
- G2_perplexity is the text file containing the perplexity values for Task2
- G1_pagerank is the text file containing the top 50 results for Task1
- G2_pagerank is the text file containing the top 50 results for Task2
- Task1_inlinks is the graph generated for G1 of Task1
- Task2_outlinks is the outlinks graph generated for G1 of Task1
- Task1_statistics is the brief report of the simple statistics over G1 and G2
- Task3_speculation is the speculation for Task3


INSTRUCTIONS TO RUN:
-------------------
PLEASE USE THE LATEST VERSION OF JAVA JRE 1.8.101 or higher.

Running the JAR file on Windows, Linux and Mac OS	
		- Download and install JRE 1.8.101 or higher.
		- Use the following command on the terminal to run java -jar Task1.jar for Task1
                - Use the following command on the terminal to run java -jar Task2.jar for Task2
